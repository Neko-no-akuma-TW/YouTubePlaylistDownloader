# YouTubeDownloader 

---

## 介紹
這是一個用於下載YouTube播放清單的Python程式碼</br>
此程式支援多線程下載，並且也同時支援下載字幕  
(不會下載自動生成的字幕與翻譯字幕)  
(僅會下載繁體中文、簡體中文、英文、日文語言的字幕)  
(優先選擇.vtt檔案，如果沒有，則會下載.srt檔案)  

---

## 使用方法
1. 安裝Python
2. 安裝ffmepg 把exe 放入 Python的 Scripts 文件夾
3. 安裝所需套件
```
pip install -r requirements.txt
```
4. 執行程式
```
python main.py
```
